## 故障排查

### 问题1：自动推送失败
- 检查网络连接
- 检查 GitHub 认证是否有效（token 或 SSH key）
- 手动运行 `git push origin master` 查看错误

### 问题2：任务计划程序没有运行
- 打开任务计划程序，查看"上次运行结果"
- 检查任务是否被禁用
- 手动右键点击任务 → "运行" 测试

### 问题3：auto_sync.py 没有提交
- 检查是否有未提交的更改：`git status`
- 手动运行 `python auto_sync.py` 查看输出
- 检查脚本是否有执行权限
