## 💡 技术实现要点

### 早报生成流程

```python
# 1. 运行搜索脚本
python run_search.py

# 2. 生成搜索结果（JSON）
briefing_search_results.json

# 3. 按 Skill 规范处理数据
- 日期过滤（T 到 T-2）
- 重要性分级（A/B/C/D）
- 三段式摘要写作
- 经营判断生成
- 行动建议推导

# 4. 输出格式
- Markdown 文档
- JSON 结构化数据
- HTML 弹窗内容
```

### 弹窗集成方式

**当前方式**: 将 HTML 内容作为 JavaScript 字符串存储在 `reportData` 对象中

```javascript
const reportData = {
    'czy': {
        title: '常州锂源',
        content: `<div class="section">...</div>`  // HTML 字符串
    }
};
```

**优点**: 简单直接，无需额外请求  
**缺点**: 文件体积大，维护困难

**替代方案**: 
- 使用外部 JSON 文件存储早报数据
- 通过 fetch 动态加载
- 模板引擎渲染
