# 磷酸铁锂行业综合洞察报告可视化看板

## 📊 项目概述

本项目基于《磷酸铁锂行业综合洞察报告_2026Q1》Word 文档中的数据，创建了 8 个交互式可视化图表和一个综合可视化看板。

## 📁 文件清单

### 核心文件
- **dashboard.html** - 综合可视化看板（主文件）
- **report_content.json** - 从 Word 文档提取的原始数据

### 图表文件（8 个）
1. **chart1_monthly_trend.html** - 月度产量/产能/利用率趋势图
2. **chart2_company_ranking.html** - 企业产量排名和利用率对比图
3. **chart3_lfp_price.html** - LFP 现货价格可视化
4. **chart4_generation_comparison.html** - 代际产品对比分析
5. **chart5_market_structure.html** - 市场份额和结构分析
6. **chart6_2025_metrics.html** - 2025 年核心数据概览
7. **chart7_2026_forecast.html** - 2026 年预测数据
8. **chart8_expansion.html** - 企业扩产项目对比

### 源代码文件
- **read_word.py** - Word 文档内容提取脚本
- **create_charts_simple.py** - 图表生成脚本

## 🚀 使用方法

### 方法 1：直接查看综合看板（推荐）
1. 双击打开 `dashboard.html` 文件
2. 在浏览器中查看所有交互式图表
3. 鼠标悬停查看详细信息

### 方法 2：单独查看图表
直接双击任意 `chart*.html` 文件查看单个图表

### 方法 3：重新生成图表
```bash
# 提取 Word 文档数据
python read_word.py

# 生成可视化图表
python create_charts_simple.py
```

## 💡 数据洞察

### 关键发现
1. **产能利用率大幅提升**：从 2024 年初的 33.5% 提升至 2025 年底的 90%
2. **行业集中度提高**：CR5 从 74.8% 提升至 80%+
3. **代际分化明显**：四代产品供不应求，二代产品加速出清
4. **储能成为新引擎**：2026 年储能电池出货量将首次超越动力电池
5. **LMFP 快速崛起**：预计 2026-2027 年占新增 LFP 体系产量的 15%-25%

### 价格趋势
- 碳酸锂：2026 年价格中枢 12-18 万元/吨
- LFP 动力型：价格中枢 5.5-6.5 万元/吨
- 代际价差：四代 vs 二代达 1,500 元/吨
