### 1.1 双服务器架构

| 端口 | 用途 | 目录 |
|------|------|------|
| **8888** | 主战场，日常修改和验证 | `D:\trae\AI Daily report\` 根目录 |
| **8089** | 嵌入版备份/汇报专用站 | `D:\trae\AI Daily report\embedded\` |

**核心规则**：
- 所有修改**必须先在 8888 验证**通过，再同步到 8089
- 同步命令：`python embedded\gen_embedded_all.py`
- 8089 的 HTML 文件将数据内嵌（`window.__EMBEDDED__`），可单文件离线运行

### 1.2 嵌入版生成脚本 (`gen_embedded_all.py`)

**作用**：将 8888 端的松散资源（JSON/JS/CSS）内嵌到 HTML 中，生成 8089 的单文件部署版本。

**数据注入对象**：
| 目标页面 | 注入变量 | 数据来源 |
|---------|---------|---------|
| `index_v3.html` | `window.__EMBEDDED__` | `reports/` 当日 JSON |
| `policy_center_v4.html` | `EMBEDDED_POLICIES` | `reports/policies.json` |
| `industry_news_embedded.html` | `EMBEDDED_NEWS` | `industry_news_source/industry_news_clean.json` |
| 各 `*_data_v2.html` | 嵌入版 | `*_embedded_data.js` 内容 |

**运行时机**：每次数据库更新后、政策累积更新后、行业新闻清洗后。

## 五、紧急故障处理流程图

### 5.1 首页无法访问

```
首页无法访问（http://localhost:8888）
  ↓
检查 localhost:8888 服务是否启动
  ↓
├─ 未启动？ → 启动服务（运行 `python -m http.server 8888`）
├─ 已启动但仍无法访问？ → 检查端口是否被占用（netstat -ano | findstr 8888）
└─ 端口被占用？ → 结束占用进程（taskkill /PID <pid> /F）
```

### 5.2 早报数据未更新

```
早报数据未更新（reports/YYYY-MM-DD.json 不存在）
  ↓
检查自动化任务是否运行（查看 .workbuddy/automations/*/memory.md）
  ↓
├─ 任务未运行？ → 手动运行任务（打开 WorkBuddy → 自动化 → 运行任务）
├─ 任务运行但失败？ → 查看 error log，参考 [05-常见错误与排障详细文档](05-常见错误与排障详细文档.md)
└─ 任务成功但数据未生成？ → 检查 MD 文件是否存在，手动运行 `D:\buddy\md_to_json.py`
```

### 5.3 数据库页面显示异常

```
数据库页面显示异常（{prefix}_data_v2.html）
  ↓
按 F12 打开浏览器控制台
  ↓
├─ JS 加载失败（404错误）？ → 检查 `embedded_data.js` 是否存在
├─ 数据过时（update_time 不是今天）？ → 参考 [02-数据库与看板运维详细文档](02-数据库与看板运维详细文档.md) 更新数据
└─ 其他错误？ → 查看控制台错误信息，搜索 [05-常见错误与排障详细文档](05-常见错误与排障详细文档.md)
```

## 六、常见错误快速索引

> **详细内容请查看**：[05-常见错误与排障详细文档](05-常见错误与排障详细文档.md)

| 错误关键词 | 可能原因 | 快速解决方案 |
|-----------|---------|---------|
| `404 (File not found)` | 文件路径错误 | 检查文件是否存在，路径是否正确 |
| `Failed to load resource` | 网络请求失败 | 检查网络连接，确认服务器已启动 |
| `sectionsToHTML returned, html.length=0` | `dim` 值不匹配 | 确保 `dim` 是英文（policy/market/competitor/frontier） |
| `buildBriefSection: no topnews section found` | 未找到今日关注section | 修改函数，从 `items` 读取 |
| 今日关注显示"暂无内容" | `section.content` 为空 | 修改 `buildBriefSection()` 读取 `items` |
| 市场/价格显示为卡片 | 未识别价格类条目 | 在 `sectionsToHTML()` 中添加 `market` 特殊处理 |
| 看板图表不更新 | 硬编码数组未更新 | 运行 `python _update_charts_data.py` 或手动编辑HTML |
| Token过期 | refresh_token失效 | 重新OAuth授权，更新 `feishu_user_token.json` |
