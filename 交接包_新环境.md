## 一、项目概述

**龙蟠行业数据库**是一个本地运行的锂电行业信息中台，包含：

1. **早报系统**：每日9个事业部（BU）的行业早报，动态加载 JSON 数据渲染
2. **分析报告**：碳酸锂、磷酸铁锂等深度分析报告（6个Tab + ECharts图表）
3. **数据库导航**：7个数据库的入口页面，含已完成/建设中状态

所有页面均为**静态 HTML**，不需要数据库，靠 `python -m http.server` 提供本地 HTTP 服务。

### 4.1 早报录入流程（每日操作）

```
1. 把9个BU的MD文件放到下载目录：

   SRC_DIR  = Path(r"C:\Users\1\Downloads\YYYY-MM-DD-9bu-reports\YYYY-MM-DD")
```

## 五、关键约束（踩过的坑）

1. **`openReport()` 里 `hy-item-body` 不能设置 inline `display`**：会导致展开/折叠逻辑失效
2. **HTML 版本号**：每次修改 HTML 后更新 `<meta name="html-version" content="YYYYMMDdx">`
3. **HTTP 服务是必须的**：部分 JS 有 `fetch('reports/xxx.json')` 动态加载，file:// 协议下会失败
4. **早报生成**：只用 `D:\buddy\md_to_json.py`，旧的 `gen_reports_from_md.py` 已废弃
5. **WebSearch/WebFetch**：当前环境有企业防火墙限制，无法访问外部网页
