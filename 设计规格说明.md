## 项目概述

### 功能定位
企业级每日早报管理系统，服务于 9 个事业部，提供：
- 当日早报查看
- 历史数据回溯
- 自动播放功能
- 多维度数据展示

### 技术栈
- **纯 HTML5 + CSS3 + JavaScript ES6**
- 无需构建工具，直接浏览器运行
- 本地 JSON 文件存储数据

### 文件结构
```
d:\trae\AI Daily report\
├── index.html                 # 主页面文件
├── index_logo_v2.html        # 带品牌 Logo 的版本（当前使用）

    └── ...
```

#### 核心变量
```javascript
let autoPlayTimer = null;          // 自动播放定时器
let currentDeptIndex = 0;          // 当前事业部索引
```

#### 控制函数
```javascript
// 开始自动播放
function startAllReports() {
    currentDeptIndex = 0;
    isPaused = false;
    openReport(deptList[0], true);
}

// 下一个事业部
function nextDept() {
    currentDeptIndex = (currentDeptIndex + 1) % deptList.length;
    openReport(deptList[currentDeptIndex], true);
}

// 上一个事业部
function previousDept() {
    currentDeptIndex = (currentDeptIndex - 1 + deptList.length) % deptList.length;
    openReport(deptList[currentDeptIndex], true);
}

// 暂停/继续
function togglePause() {
    isPaused = !isPaused;
    // 更新按钮文字和状态
}

// 停止播放
function stopAutoPlay() {
    clearInterval(autoPlayTimer);
    clearInterval(progressTimer);
    // 清理进度条
}
```

#### 进度条
```javascript
// 顶部进度条，宽度 4px，渐变色
let progress = 0;
progressTimer = setInterval(() => {
```

### 2. 弹窗管理系统

#### 打开早报弹窗
```javascript
function openReport(deptId, autoPlay = false) {
    const data = reportData[deptId];
    if (!data) return;
    
    // 设置标题和日期
    document.getElementById('modalTitle').textContent = data.title;
    document.getElementById('modalDate').textContent = 
        data.subtitle + ' | 2026 年 4 月 3 日';
    
    // 加载内容
    document.getElementById('modalBody').innerHTML = data.content;
    
    // 显示弹窗
    document.getElementById('reportModal').classList.add('active');
    document.body.style.overflow = 'hidden';
    
    // 如需自动播放
    if (autoPlay) {
        startAutoPlay();
    }
}
```

#### 关闭弹窗
```javascript
function closeReport() {
    stopAutoPlay();
    document.getElementById('reportModal').classList.remove('active');
    document.body.style.overflow = 'auto';
}

// 点击外部关闭
document.getElementById('reportModal').addEventListener('click', function(e) {
    if (e.target === this) {
        closeReport();
    }
});

// ESC 键关闭
document.addEventListener('keydown', function(e) {
    if (e.key === 'Escape') {
        closeReport();
    }
});
```

### API 端点

#### 1. 获取索引
```

```

#### 2. 获取每日数据
```
GET /reports/{date}.json
Response: 该日期的完整数据
```

### 数据生成脚本（Python 示例）

```python
import json
from datetime import datetime, timedelta

def generate_daily_report(date, departments):
    """生成单日报表 JSON"""
    report = {
        "date": date,
        "departments": {},
        "categories": {
            "policy": {"name": "政策", "icon": "📜", "items": []},
            "competitor": {"name": "竞品", "icon": "🏆", "items": []},
            "customer": {"name": "客户", "icon": "👥", "items": []},
            "technology": {"name": "技术", "icon": "🔬", "items": []}
        },
        "summary": {
            "total_news": 0,
            "hot_topics": [],
            "market_trend": "平稳",
            "key_insight": ""
        }
    }
    
    # 填充各事业部数据
    for dept in departments:
        report["departments"][dept["code"]] = {
            "name": dept["name"],
            "subtitle": dept["subtitle"],
            "sections": [],
            "charts": []
        }
    
    # 保存到文件
    with open(f'reports/{date}.json', 'w', encoding='utf-8') as f:
        json.dump(report, f, ensure_ascii=False, indent=2)
    
    return report

def update_index():
    """更新索引文件"""
    import os
    dates = [f.replace('.json', '') for f in os.listdir('reports') 
             if f.endswith('.json') and f != 'index.json']
    dates.sort(reverse=True)
    
    index = {
        "available_dates": dates,
        "latest_date": dates[0] if dates else None,
        "total_reports": len(dates)
    }
    
    with open('reports/index.json', 'w', encoding='utf-8') as f:
        json.dump(index, f, ensure_ascii=False, indent=2)

# 使用示例
departments = [
    {"code": "lhy", "name": "润滑油", "subtitle": "Lubricant Division"},
    {"code": "kls", "name": "可兰素", "subtitle": "Kelasen Division"},
    # ... 其他事业部
]

generate_daily_report('2026-04-08', departments)
update_index()
```

## 部署指南

### 1. 环境要求
- **Web 服务器**: 任何静态文件服务器（Nginx, Apache, IIS 等）
- **浏览器**: 现代浏览器（Chrome 80+, Firefox 75+, Edge 80+）
- **JavaScript**: ES6 支持
- **无需后端**: 纯前端应用

### 2. 部署步骤

#### Step 1: 文件准备
```bash
# 创建项目目录
mkdir -p "AI Daily report/reports"

# 复制主文件
cp index_logo_v2.html "AI Daily report/index.html"

# 复制示例数据
cp reports/*.json "AI Daily report/reports/"
```

#### Step 2: 配置 Web 服务器

**Nginx 配置示例:**
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    root /path/to/AI Daily report;
    index index.html;
    
    location / {
        try_files $uri $uri/ =404;
    }
    
    # JSON 文件 MIME 类型
    location ~ \.json$ {
        add_header Content-Type application/json;
    }
}
```

**Apache 配置示例:**
```apache
<VirtualHost *:80>
    ServerName your-domain.com
    DocumentRoot /path/to/AI Daily report
    
    <Directory /path/to/AI Daily report>
        Options Indexes FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
```

#### Step 3: 数据初始化

## 常见问题

### Q1: 历史数据无法加载
**检查**: 
1. `reports/index.json` 是否存在
2. JSON 格式是否正确
3. 文件路径是否正确

### Q2: 自动播放不工作
**检查**:
1. 浏览器是否允许自动播放
2. JavaScript 控制台是否有错误
3. `startAllReports()` 函数是否正确调用

### Q3: Logo 颜色不显示
**检查**:
1. `style="background: #COLOR;"` 是否正确
2. CSS 是否被覆盖
3. 浏览器开发者工具检查样式
